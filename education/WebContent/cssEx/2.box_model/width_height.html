height 속성

height - height는 요소의 높이 지정하는 속성이다. display속성이 “inline”인 요소에는 적용되지 않으며 height 속성의 적용되는 방식은 box-sizing 속성값에 따른다. 콘텐츠 높이가 지정된 값보다 커도 요소의 높이는 변경되지 않는다. 음수 값은 지정 할 수 없다.
